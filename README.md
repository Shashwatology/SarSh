# Sarsh - Private Messaging Platform

Sarsh is a modern Indian messaging platform focused on privacy and real-time communication.

## ✨ Features
- **Real-time Messaging**: Instant message delivery via Socket.IO.
- **Privacy First**: Secure authentication and private chats.
- **Media Sharing**: Image, audio, and document uploads.
- **Message Reactions**: Express yourself with emojis on messages.
- **Presence Tracking**: Online/Away status and Last Seen indicators.
- **WebRTC Calling**: Integrated voice and video calls.
- **Theme Customization**: Multiple UI themes including Dark and Modern modes.

## 🚀 Getting Started

### Prerequisites
- Node.js & npm
- Docker (for PostgreSQL)

### Backend Setup
1. `cd chat-app/backend`
2. `npm install`
3. Create a `.env` file based on the environment guide.
4. `npm run dev`

### Frontend Setup
1. `cd chat-app/frontend`
2. `npm install`
3. `npm run dev`

## 🛠️ Technology Stack
- **Frontend**: Next.js, Tailwind CSS, Lucide React
- **Backend**: Express.js, Socket.IO, PostgreSQL
- **Database**: PostgreSQL (Dockerized)
- **External**: Cloudinary (Media), Web Push API (Notifications)
